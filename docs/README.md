# 📚 Jooblie Documentation

This folder contains documentation for external developers and integrations.

## 📁 Contents

### `/flutter-api/`

API documentation for Flutter mobile app developers. These files are **NOT** part of the React web application and should **NOT** be committed to git.

**Files:**

- `FLUTTER_API_DOCUMENTATION.md` - Complete API reference
- `FLUTTER_CODE_EXAMPLES.dart` - Dart code examples
- `API_QUICK_REFERENCE.md` - Quick reference guide

**Usage:**
Share these files directly with Flutter developers who need to integrate with the Jooblie backend API.

---

## ⚠️ Important Notes

1. The `flutter-api` folder is ignored in `.gitignore`
2. These are reference documents only, not part of the codebase
3. Update these docs when backend APIs change
4. Do not include these files in production builds
